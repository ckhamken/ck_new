Project Overview

This project comprised two phases, each consisting of three separate robotic assembly stations installed at various stages of an EV drive unit assembly line — six stations in total. They replaced existing manual operator stations in order to reduce headcount while increasing throughput consistency and traceability.

Each station is tasked with installing one to two components into a drive unit located on a work pallet shuttled along a conveyor loop. Stations 1 and 3 receive component material via tote stacks along an over-under roller conveyor. Station 2 is supplied material by two dial tables consisting of component stacks. All material infeed is verified with barcode scanners on either totes or packaging boxes.

Each robot picks components from their respective material supplies, transfers them to an inspection module to verify quality compliance, then proceeds to installation. Where lubricant application is required, the robot performs this with an integrated spray nozzle on the end-of-arm tool. Installation at each station is monitored by insertion forces, vision presence checks, and fastening torques.

Project Structure
Two phases, six stations, one production line

The project was structured across two delivery phases to allow for staged installation alongside active production. Each phase delivered three complete robotic assembly stations, with the second phase building on lessons learned from Phase 1 commissioning. Total scope management included coordination with the customer's production engineering team throughout FAT qualification and final site installation.
