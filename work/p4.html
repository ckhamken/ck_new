Project Overview

A stand-alone station to supplement production capacity of armatures for electric motors. Operator loads a tray of armatures onto a holding nest. Pins are auto-fed by a vibratory bowl feeder. Upon cycle start the robot picks an armature from a sequential tray position and transfers it to process module 1, where a pin is blow-fed, positioned, and pressed to a hard stop — establishing the pin protrusion from the armature. The assembled armature is then transferred to process module 2 for runout measurement: the pin is positioned and clamped onto a roller assembly, a servo motor rotates the rollers, and an optical micrometer scans a section of the armature body to determine runout relative to the pin. Acceptable runout is ±0.02mm. NOK assemblies go to a reject chute; OK assemblies return to an open tray position.

Process Modules
Press then measure — two operations, one station

Process module 1 establishes pin protrusion via a hard-stop press. Process module 2 measures runout of the pressed pin relative to the armature body using an optical micrometer while servo-driven rollers rotate the assembly. The two-module layout keeps the robot utilization high and the cycle time to 9 seconds.
