Project Overview

The objective of this project is to increase the production rate of an existing manual operation tasked with packaging test kits. A packaged test kit assembly consists of a test kit unit and an information pamphlet placed into a cardboard carton that is folded shut. The machine operates in three sections: material infeed handling, carton insertion and processing, and outfeed handling. In the infeed area, an operator manually loads three feed lanes: cartons (presented flat and automatically formed), test kit units (inspected for correct orientation before label application and carton insertion), and information cards (pre-folded, fed directly on top of the inserted test kit unit). Cartons then proceed through sequential stations that tuck, fold, and close the top cover. During outfeed, a label is applied to the carton exterior, verified, and the completed assembly is sorted to OK or NOK outfeed lanes.

Machine Sections

The three-section machine design separates material presentation, carton processing, and outfeed verification into distinct zones. This modular approach simplified debugging during FAT and allowed each section to be validated independently before full-line integration.
